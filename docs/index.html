About The Project

Welcome to the showcase of our 3D localization and visualization system. This project aims to provide real-time tracking and environmental awareness for emergency responders. Below is the story of how this system came to life, from the first lines of code to a fully functional 3D interface.

Phase 1: The Foundation

Every great project starts with a blank canvas. Our first build was humble—a simple 3D view populated with randomly spaced balls to establish our coordinate system and test the basic rendering engine.

The initial 3D view: Randomly spaced spheres setting the stage for what was to come.

Phase 2: Entering the Third Dimension

The core of our project is 3D localization. We began rendering buildings and structures in 3D, allowing for vertical tracking—crucial for multi-story operations.

3D Localization: Adding depth and verticality to the tracking.

Phase 3: Building the UI and Agents

With the 3D world established, we focused on the interface. We introduced firefighter icons and a dashboard displaying critical data such as BPM and status, visualizing the agents within the system.

Early interface showing firefighter icons and vital statistics like BPM and state.

Phase 4: Tracking Movement

Static points are useful, but movement tells a story. We implemented trajectory tracking to visualize the path taken by agents over time.

Early trajectory testing.
Refined trajectories: Smooth and informative paths.

Phase 5: Polishing the Experience

As we neared completion, we focused on the user interface, ensuring that critical information is easily accessible without cluttering the view.

The near-final interface: Clean, functional, and powerful.
